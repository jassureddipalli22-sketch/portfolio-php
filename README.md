# Portfolio Website with PHP & MySQL

Dynamic portfolio site with contact form and admin panel.

## Features
- Responsive portfolio design
- Contact form that saves to MySQL database  
- Admin page to view/delete messages
- CSS styling and form validation

## Tech Stack
- PHP
- MySQL 
- HTML/CSS

## How to Run Locally
1. Install XAMPP
2. Copy files to `htdocs/portfolio/`
3. Start Apache + MySQL in XAMPP
4. Import database: Create `portfolio_db` and `messages` table
5. Open `http://localhost/portfolio/`
